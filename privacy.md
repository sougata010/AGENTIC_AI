# Privacy Policy

**Last Updated: [Date]**

This Privacy Policy describes how the "Agentic AI Image Generator" project handles information.

## Information Collection and Use

This project is an open-source automation tool designed to run locally on your machine.
*   **No Data Collection:** We (the repository owners) do not collect, store, or process any personal data from users of this software.
*   **Local Operation:** All operations, including trend analysis and image generation, occur locally or via direct storage on your device.
*   **Third-Party Services:** The tool interacts with third-party APIs (e.g., Google Gemini, OpenAI). Please refer to their respective privacy policies to understand how they handle data you send to them via your API keys.

## API Keys

*   Your API keys (`GOOGLE_API_KEY`, `OPENAI_API_KEY`) are stored locally in your `.env` file.
*   They are **never** transmitted to us or any unauthorized third party.

## Changes to This Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this repository.

## Contact Us

If you have any questions about this Privacy Policy, please open an issue in the GitHub repository.
